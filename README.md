# Groq Chatbot with Conversation History

A Streamlit-based chatbot built with **Groq** and **LangChain** that supports **conversation history**, understands **follow-up questions**, and provides **context-aware responses**.

## Features

- Fast chatbot responses using **Groq**
- Built with **LangChain**
- Maintains **conversation history**
- Handles **follow-up questions contextually**
- Simple and interactive **Streamlit chat interface**
- Uses proper **LangChain message history**
- Easy to run locally and deploy on Streamlit Cloud

## Tech Stack

- Python
- Streamlit
- Groq
- LangChain
- python-dotenv
